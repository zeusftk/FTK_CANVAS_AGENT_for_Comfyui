
# FTK_CANVAS_AGENT 项目介绍
[ENGLISH README](https://github.com/zeusftk/FTK_CANVAS_AGENT_for_Comfyui/blob/main/README_EN.md)
## 项目概述  
FTK Canvas Agent是一款功能强大的多媒体处理与自动化工具集，专注于视频、音频处理及智能剪辑。  

该工具集成多项先进AI技术，覆盖视频、音频、剪辑全流程处理，核心能力包括 **✨ AI剪辑、AI特效、智能配音、AI自动评论** 等。其内置横竖屏分类、场景切片、动作片段提取、字幕添加、音色转换等实用功能，支持 **🚀 开箱即用零配置**。  

FTK Canvas Agent 核心功能：通过连接ComfyUI实现AI服务，智能管理ComfyUI工作流，实现仅靠通过对话无感调用工作流完成各种任务。

## 核心功能

## 核心功能：智能ComfyUI工作流管理

FTK Canvas Agent 配置 ComfyUI 后，可通过聊天自动调用并执行 ComfyUI workflow，更能帮 workflow 管理者打通 “设计 - 分发 - 服务” 全链路，让优质 ComfyUI workflow 持续释放价值、触达更多需求场景。

### 🗣️ 智能对话驱动：降低门槛，扩大 ComfyUI workflow 覆盖
- 自然语言控制：用户无需复杂操作，聊天即可触发你的 ComfyUI workflow，轻松触达更多潜在使用者
- 智能参数调节：系统自动适配 ComfyUI workflow 参数，减少沟通成本，提升 workflow 使用体验
- 零基础友好：面向无技术背景群体，让你的 ComfyUI workflow 突破场景限制，覆盖更广泛需求

### ⚙️ 专业 workflow 管理：强化权限，保障 ComfyUI workflow 价值
- 专家级工具支持：提供 ComfyUI workflow 完整管理、加密分发能力，防止设计外泄，守护 workflow 核心价值
- 智能化调用：按用户需求精准匹配 ComfyUI workflow，提升使用频次，让设计持续发挥作用
- 手动指定调用：支持限定 ComfyUI workflow 使用范围，可按场景 / 人群灵活划分服务模式，适配多样需求

### 🌐 自动化服务：简化流程，提升 ComfyUI workflow 运营效率
- 自动 workflow 调用：用户指令触发后，系统自动执行 ComfyUI workflow，无需手动介入，降低服务成本
- 参数智能映射：精准转化需求为 ComfyUI workflow 参数，减少执行误差，提升用户满意度与复购意愿
- 结果自动返回：ComfyUI workflow 执行结果直传用户，简化交付链路，助力高效服务更多用户、拓展 workflow 服务规模

### 适用人群
#### 🎯 零基础用户
- 简单易用：通过自然语言对话完成复杂图像生成任务
- 无需技术背景：不需要了解ComfyUI或AI模型技术细节
#### 👨‍💻 工作流专家
- 完整控制：提供ComfyUI的完整功能和专业控制能力
- 工作流分发：创建、管理和分发自定义工作流给团队使用


## 其他功能
### 💬 智能对话
- **多轮对话**: 支持上下文理解的智能聊天
- **创意辅助**: 协助用户进行创意思考和文案写作
- **知识问答**: 基于大语言模型的知识查询
- **多语言支持**: 支持多种语言的对话交互

### 🎨 AI图像生成（需下载内置模型包）
- **文本到图像**: 通过自然语言描述生成高质量图像
- **图像风格化**: 将普通图片转换为艺术风格作品
- **图像修复与增强**: 智能修复模糊或损坏的图像
- **图像扩展**: 智能填充和扩展图像边界

### 🎬 视频创作（需下载内置模型包）
- **文本生成视频**: 通过自然语言描述生成动态视频
- **图像转视频**: 将静态图像转化为动态视频
- **视频风格迁移**: 为现有视频添加艺术效果

## 常见问题

### 启动问题
**Q: 程序无法启动怎么办？**
A: 检查data目录完整性，确认系统满足最低配置要求

**Q: 程序显示：任务执行中，请稍后**
A: 查看黑色终端如果没有报错，就是还在运行
时间很久有两种原因：
1.第一次使用，内部部分模型需要编译为tensorrt的，为提高运行速度。
2.显卡性能制约，如RTX3060等，建议使用RTX3060以上显卡。
建议使用配置的comfyui使用自定义工作流，可以看到每个节点的进度。
后续会改进内置推理器，添加推理进度显示。

**Q: GPU加速不工作如何解决？**
A: 确认是否是NVidia显卡，驱动更新到最新

### 性能问题
**Q: 处理速度很慢怎么优化？**
A: 关闭其他占用资源的程序，升级硬件配置

**Q: 内存不足如何解决？**
A: 增加虚拟内存，关闭不必要的应用程序
